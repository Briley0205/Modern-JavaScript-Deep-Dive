### 프로그래밍 📑

+ 컴퓨터가 이해할 수 있도록 정확하고 상세하게 설명 해야한다. (상대적인 개념 -> 양적 개념 '숫자')
+ 프로그래밍 언어를 통해 컴퓨터에게 요구(명령) 하는 등의 커뮤니케이션이 프로그래밍이다. (약속된 구문 'syntax' & 의미 'semantics')
+ 프로그래밍 언어도 또한 언어임으로 문법이 중요하지만, 의미 또한 가지고 있어야 언어의 역할을할 수 있다.
