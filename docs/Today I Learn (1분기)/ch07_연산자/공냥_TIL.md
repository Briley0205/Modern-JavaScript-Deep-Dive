### ✨**자바스크립트의 다양한 연산자**✨

연산의 대상을 피연산자라고 하며, 이 피연산자는 **값으로 평가될 수 있는 표현식**이어야 한다. <br>

➡️**산술연산자**<br>
수학적 계산을 수행하는 연산자. 산술 연산이 불가능한 경우 NaN을 반환한다. 피연산자의 개수에 따라 단항산술연산자, 이항산술연산자로 나눌 수 있다.<br>
| 단항산술연산자 | 이항산술연산자 |
| :-------: | :-----------: |
| `+` `-` `*` `/` `%` | `++` `--` `+` `-` |

이항산술연산자 중 `++`, `--`는 위치에 따라 연산 방식이 달라진다. <br>
❕**전위증가/감소 연산자** <br>
피연산자의 값을 먼저 증감 시킨 후 다른 연산을 수행한다.

❕**후위증가/감소 연산자** <br>
다른 연산을 수행 후, 피연산자의 값을 증감시킨다.

❕**문자열 연결 연산자** <br>
`+`연산자는 피연산자 중 하나 이상이 문자열이라면 문자열 연결 연산자로 동작한다.

<br>

➡️ **할당연산자** <br>
피연산자의 평가 결과를 변수에 할당하는 연산자.

➡️ **비교연산자** <br>
if나 for문 같은 제어문 조건에서 종종 쓰이는 연산자. 동등/일치 비교 연산자와 대소 관계 비교 연산자가 있다. <br>

❕**동등/일치 비교 연산자** <br>
| 비교 연산자 | 설명 |
| :-------: | :-----------: |
| == | 동등 비교, 값만 비교 |
| === | 일치 비교, 값&타입 비교 |
| != | 부동등 비교, 값만 비교 |
| !== | 불일치 비교, 값&타입 비교 |

<br>

❕**대소 관계 비교 연산자** <br>

```
x > y
x < y
x >= y
x <= y
```

<br>

➡️**삼항 조건 연산자** <br>
`조건식? 조건식이 true일 때 반환할 값 : 조건식이 false 일때 반환할 값`

```
let result = score>80 ? 'pass', 'fail';
//result가 80을 넘는 경우(ture) 'pass'가 되고 그렇지 못한 경우 (false)는 'fail'이 된다.
```

<br>

➡️**논리 연산자** <br>

- || : 논리합, or 둘 중 하나라도 true면 true
- && : 논리곱, and 둘 중 하나라도 false면 false
- ! : 부정, not

<br>

➡️**쉼표연산자** <br>
왼쪽 피연산자부터 차례대로 평가한 다음 마지막 피연산자 평가가 끝나면 마지막 피연산자 평가 결과를 반환한다.

```
var a, b, c;
a = 1, b = 2, c = 3; / 3
```

➡️**그룹연산자** <br>
소괄호 () 로 피연산자를 감쌀 경우 가장 먼저 평가한다. **그룹 연산자는 연산자 우선순위 중 가장 높다.**
<br>

➡️**typeof 연산자** <br>
피연산자의 데이터 타입을 문자열로 반환한다.
<br>

➡️**지수연산자** <br>
좌항의 피연산자를 밑으로, 우항의 피연산자를 지수로 거듭제곱해서 숫자 값을 반환한다.
`2 ** 3; // 8`
지수 연산자 도입 전에는 Math.pow를 사용했다. 지수 연산자는 Math.pow 메서드보다는 가독성이 더 좋다.

### 책속 한구절 보관함 📖

> p. 78<br>
> 개발자의 의도와 무관하게 자바스크립트 엔진에 의해 암뭄적으로 타입이 자동 변환되기도 한다. 이를 암묵적 타입 변환(implict coercion) 또는 타입 강제 변환(type coercion)이라고 한다.

> p. 85 <br>
> 조건에 따라 어떤 값을 결정할 경우 if...else 문보다는 삼항 조건 연산자 표현식을 사용하는 게 유리하다. 하지만 조건에 따라 수행해야 할 문이 하나가 아닌 여러 개라면 if...else문의 가독성이 더 좋다.

> p. 88 <br>
> 값이 null타입인지 확인할 때는 typeof 연산자를 사용하지 말고 일치 연산자(===)를 사용하자.
