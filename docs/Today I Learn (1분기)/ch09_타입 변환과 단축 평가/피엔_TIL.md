### 궁금한 내용이나, 더 알아보고 싶은 내용을 적어봅시다 🤔
```
1 + "2" 의 결과값은 12다. 숫자1이 문자열로 암묵적으로 문자로 변환되어 "1" + "2"가 된 것이다. 
그런데  1 - "2" 의 결과값은 -1이다. 문자"2"가 숫자로 변환되어 1 - 2 로 계산이 된 것이다. 
내 생각에 둘다 산술 연산자인데 왜 +를 사용한쪽은 문자열로 변환되고 -를 사용한쪽은 숫자열로 변환되는지 더 알아보고 싶다.
```
