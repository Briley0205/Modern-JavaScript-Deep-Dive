### 식별자의 영향권을 결정짓는 범위, 스코프 📑

- 식별자는 자신을 참조할 수 있는 범위를 가진다.
- 같은 폴더 내에서 동일한 이름을 사용할 수 없듯이,</br> 같은 스코프 내에서 동일한 식별자 이름을 사용할 수 없다.
- 스코프는 하나의 계층적 구조(스코프 체인)를 이루고 있으며,</br> JS 엔진은 이를 통해 선언된 변수를 검색한다.

---

### 스코프의 활동 루틴

스코프는 모든 식별자의 참조 범위를 결정짓는다.</br>
이는 식별자의 선언 위치에 따라 달라진다.</br>
`> 함수의 매개변수` : 해당 함수의 몸체 내부</br>
`> 식별자` : 소스 코드의 문맥에 따라 상이함</br>

|  | 전역 스코프 | 지역 스코프 |
|----|----|----|
| 변수 | 전역 변수 | 지역 변수 |
| 참조 범위 | 어디서든 참조 가능 | 함수 내부에서 참조 |
| 코드 위치 | 코드의 가장 바깥 부분 | 함수 몸체 내부 |
