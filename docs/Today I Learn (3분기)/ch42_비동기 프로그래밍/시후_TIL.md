### 싱글 스레드 비동기 방식 프로그래밍 📑

- 자바스크립트는 원래 동기적으로 처리하는 언어이다.
- 하지만 퍼포먼스 향상을 위해 브라우저 엔진은<br> 비동기적인 처리기능을 보여주기도 한다.

---

### 동기 / 비동기

> 동기 처리

: 현재 실행 중인 태스크가 종료할 때까지<br>
다음에 실행될 테스크가 대기하는 방식

> 비동기 처리

: 현재 실행 중인 태스크가 종료되지 않아도<br>
다음 태스크를 곧바로 실행하는 방식
